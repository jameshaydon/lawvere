Quick start

  • Type an expression: 40 + 2 or "hello".
  • :l examples/tutorial.law — load a bundled example.
  • :l README.md — the README is itself a runnable literate Lawvere file.
  • :l (no argument) — open a local .law from your machine.
  • :r reloads the current file. :q quits.
